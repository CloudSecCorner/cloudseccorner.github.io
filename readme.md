# Tools 

## Application Security
- [OWASP](https://owasp.org/) - The Open Worldwide Application Security Project is dedicated to improving software security with tools, documents, and communities.

## Penetration Testing
- [Metasploit](https://www.metasploit.com/) - Advanced open-source penetration testing platform that enables you to find, exploit, and validate vulnerabilities.

## Security Distribution
- [Kali Linux](https://www.kali.org/) - Linux distribution designed for digital forensics and penetration testing with hundreds of pre-installed tools.

## Network Security
- [Nmap](https://nmap.org/) - Network mapper that discovers hosts and services on a computer network by sending packets and analyzing responses.
- [Snort](https://www.snort.org/) - Open-source network intrusion prevention and detection system capable of real-time traffic analysis.

## Network Analysis
- [Wireshark](https://www.wireshark.org/) - Network protocol analyzer that lets you see what's happening on your network at a microscopic level.

## Web Security
- [Burp Suite](https://portswigger.net/burp) - Integrated platform for performing security testing of web applications with various tools for the entire testing process.
- [OWASP ZAP](https://www.zaproxy.org/) - Open-source web application security scanner for finding vulnerabilities in web applications.
- [Nikto](https://cirt.net/Nikto2) - Open-source web server scanner that performs comprehensive tests against web servers for multiple vulnerabilities.
- [W3af](https://github.com/andresriancho/w3af) - An open-source web application security scanner that helps secure web applications by finding and exploiting all web application vulnerabilities.
- [Arachni](https://github.com/Arachni/arachni) - A feature-rich, modular, high-performance Ruby framework aimed towards helping penetration testers and administrators evaluate web application security.
- [Wapiti](https://github.com/wapiti-scanner/wapiti) - A web application vulnerability scanner that performs black-box scans to audit the security of web applications.
- [Vega](https://github.com/subgraph/Vega) - An open-source web security scanner and testing platform to test the security of web applications.

## Intrusion Detection
- [OSSEC](https://www.ossec.net/) - Open-source host-based intrusion detection system that performs log analysis, integrity checking, and more.

## Vulnerability Scanner
- [OpenVAS](https://www.openvas.org/) - Open-source vulnerability scanner and manager for detecting security issues in systems and web applications.
- [Nuclei](https://github.com/projectdiscovery/nuclei) - A fast and customizable vulnerability scanner based on simple YAML-based DSL.
- [Qualys VMDR](https://www.qualys.com/apps/vulnerability-management-detection-response/) - Combines vulnerability scanning with AI-driven risk prioritization, ideal for large IT infrastructures.

## System Auditing
- [Lynis](https://cisofy.com/lynis/) - Security auditing tool for systems running Linux, macOS, or Unix-based operating systems.

## Web Application Firewall
- [ModSecurity](https://modsecurity.org/) - Open-source web application firewall (WAF) that monitors HTTP traffic and protects against attacks.

## Password Cracking
- [Hashcat](https://hashcat.net/hashcat/) - World's fastest and most advanced password recovery utility supporting over 300 hash types.

## Wireless Security
- [Aircrack-ng](https://www.aircrack-ng.org/) - Network software suite for assessment of WiFi network security with monitoring, testing, and attacking capabilities.

## Reconnaissance
- [Shodan](https://www.shodan.io/) - Search engine for Internet-connected devices, allowing searches for specific types of computers and services.

## Cloud Security
- [Falco](https://falco.org/) - Cloud-native runtime security tool designed to detect anomalous activity in containerized applications.

## Container Security
- [Trivy](https://github.com/aquasecurity/trivy) - Comprehensive vulnerability scanner for container images, file systems, and Git repositories.

## Vulnerability Management
- [DefectDojo](https://www.defectdojo.org/) - Open-source vulnerability management tool that streamlines the testing process and offers metrics and reporting.

## Threat Intelligence
- [MISP](https://www.misp-project.org/) - Open-source threat intelligence platform for sharing, storing, and correlating IOCs and threat intelligence.

## Security Guidelines
- [OWASP Top Ten](https://owasp.org/www-project-top-ten/) - A standard awareness document for developers and web application security, representing broad consensus about the most critical security risks to web applications.
- [OWASP ASVS](https://owasp.org/www-project-application-security-verification-standard/) - The OWASP Application Security Verification Standard provides a framework for testing web application security controls.

## DAST
- [OWASP ZAP](https://github.com/zaproxy/zaproxy) - A full-featured, free, and open-source DAST tool that includes both automated scanning for vulnerabilities and tools for manual web app penetration testing.
- [Acunetix](https://www.acunetix.com) - A commercial tool focusing on web application vulnerability scanning, identifying issues such as SQL injection and XSS.

## Web Server Security
- [Nikto](https://github.com/sullo/nikto) - An open-source web server scanner which performs comprehensive tests against web servers for multiple items, including over 6700 potentially dangerous files/CGIs.

## XSS Detection
- [XSStrike](https://github.com/s0md3v/XSStrike) - The most advanced XSS detection suite with a powerful fuzzing engine and context analysis engine.

## SQL Injection
- [Sqlmap](https://github.com/sqlmapproject/sqlmap) - An open-source penetration testing tool that automates the process of detecting and exploiting SQL injection flaws.

## Training
- [SANS Institute Training](https://www.sans.org) - Comprehensive cybersecurity training offering more than 60 courses across all practice areas including web security.
- [Web Security Academy](https://portswigger.net/web-security) - A free online training center for web application security, including content from PortSwigger's in-house research team.
- [TCM Security Academy](https://academy.tcm-sec.com) - Practical, job-focused cybersecurity training designed by industry-leading instructors.
- [Cybrary](https://www.cybrary.it) - Online cybersecurity courses and training for individuals and teams, including certification preparation.

## Security Framework
- [OWASP ASVS](https://owasp.org/www-project-application-security-verification-standard/) - The OWASP Application Security Verification Standard provides a framework for testing web application security controls.

## API Security
- [OWASP API Security Top 10](https://owasp.org/www-project-api-security/) - A standard reference guide highlighting the most critical web API vulnerabilities.

## Learning Resources
- [Paragonie Awesome AppSec](https://github.com/paragonie/awesome-appsec) - A curated list of resources for learning about application security, including books, websites, blog posts, and self-assessment quizzes.

## AI Agent
- [bumpgen](https://github.com/xeol-io/bumpgen) - An AI agent that upgrades npm packages.
